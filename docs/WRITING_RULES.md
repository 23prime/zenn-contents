# Zenn 記事執筆ルール

このファイルは Zenn 記事を書く際の共通ルールを定義する。
執筆スキル・レビュースキルの両方から参照される。

## 記事構成

- 本文内の見出しは h2〜h4 のみ使用する（h1 は記事タイトル用）
- 各セクション見出し（h2〜h4）の先頭に、内容にフィットする絵文字を1つ添える
- 段落間は空行で区切る（単一の改行は無視される）
- HTML タグは非サポート（`<br>` を除く）

## 半角・全角ルール

| 対象 | ルール |
| --- | --- |
| 英数字 | 半角 |
| 英数字に付随する記号（区切り文字 `-` `_` `+`、単位 `$` `%`、パス `/` `\`、その他 `#` `@` `&` `=` `~` `^` `\|` `<` `>`） | 半角 |
| コードブロック・インラインコード内の記号 | すべて半角（そのまま） |
| 日本語文中の句読点・括弧（`！` `？` `：` `；` `（）` `「」` `【】` `『』` `…`） | 全角 |
| 日本語文中の引用 | `「」` を使う（`"` `'` は使わない） |
| ダッシュ | 日本語文中では原則使わない。どうしても必要な場合は半角 `--` |
| 範囲を示す波線 | 日本語文中では全角 `〜`、コード中では半角 `~` |

## カタカナ語の表記

[外来語の表記 留意事項その2（文化庁）](https://www.bunka.go.jp/kokugo_nihongo/sisaku/joho/joho/kijun/naikaku/gairai/honbun06.html) に準拠する。

- 長音は原則として長音符号「ー」を用いて書く
- 英語の語末 -er / -or / -ar に当たるものは原則としてア列の長音とし「ー」を用いる。ただし慣用に応じて省くことができる
  - 「ー」あり：サーバー、ユーザー、メンテナー、コンピューター、エレベーター
  - 慣用により「ー」なし：コンテナ、ディレクトリ、フォルダ

## 禁止事項

- 特定の国・政府・地政学的な陰謀への言及
